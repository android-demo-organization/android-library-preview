# Android-Library-Preview

一个用来查看图片、播放视频、播放音频的库

## gradle

project

    allprojects {
        repositories {
            ...
            ...
            ...
            maven {
                url "https://jitpack.io"
            }
        }
    }
    
    
module

    implementation 'com.github.HyFun:Android-Library-Preview:1.0.0'
